This application was created as a part of engineering thesis at AGH University of Kraków.
Data was taken from Gmina Miejska Kraków, Portal MSIP Obserwatorium and Geofabrik.
During the development of this application, the following frameworks and technologies were used:

Frontend:

  • Angular 17
  • Angular Material
  • Openlayers
    • CartoDB
    • OpenStreetMap
    • Proj4.js
  • Tailwind CSS
  • ng-circle-progress
  • Toastr

Backend:

  • .NET 8
    • Entity Framework Core
    • Npgsql with NetTopologySuite
  • xUnit
  • PostgreSQL with PostGIS extension
